# Quotes Recommendation Chatbot Using NLP

This project is an NLP-based chatbot built using Rasa that recommends motivational quotes based on user input.

## Features

- NLP-based intent recognition
- Motivational quote recommendations
- Emotional support responses
- Interactive chatbot

## Technologies Used

- Python
- Rasa
- NLP
- YAML

## Run the Project

Install dependencies:

pip install rasa

Train model:

rasa train

Run chatbot:

rasa shell
